# 🛠 Selenium & Pytest 학습 저장소

이 저장소는 셀레니움 기초부터 `pytest`를 활용한 테스트 자동화까지의 학습 과정을 기록합니다.

---

## 📜 Git 커밋 컨벤션 (Commit Convention)

본 프로젝트는 일관된 코드 이력 관리를 위해 아래의 커밋 규칙을 준수합니다.

| 타입 (Type) | 내용 (Description) |
| :--- | :--- |
| **feat** | 새로운 기능 추가 또는 기존 기능 요구 사항 수정 |
| **fix** | 버그 수정 |
| **build** | 빌드 관련 수정, 모듈 설치 및 삭제 (pip 등) |
| **chore** | 패키지 매니저 설정, `.gitignore` 등 기타 잡무 |
| **ci** | CI(GitHub Actions 등) 관련 설정 수정 |
| **docs** | 문서 수정 (README.md, 주석 등) |
| **style** | 코드 포맷팅, 세미콜론 누락 등 스타일 수정 (로직 변경 X) |
| **refactor** | 코드 리팩터링 (변수명 변경 등 로직 구조 개선) |
| **test** | 테스트 코드 추가 및 수정 |
| **release** | 버전 릴리즈 |

### 💡 커밋 메시지 예시
- `feat: 구글 로그인 버튼 클릭 자동화 스크립트 추가`
- `docs: 요소 찾기 학습 내용 리드미 업데이트`
- `build: pytest 라이브러리 설치`
